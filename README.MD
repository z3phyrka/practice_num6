# Система управления онлайн-магазином

Проект демонстрирует применение архитектурных шаблонов и паттернов проектирования.

## Используемые технологии и паттерны

### Архитектурные шаблоны:
- MVC (Model-View-Controller)
- Микросервисная архитектура
- API Gateway

### Паттерны проектирования:
1. **Структурные:**
   - Adapter
   - Facade

2. **Поведенческие:**
   - Observer
   - Strategy

3. **Порождающие:**
   - Singleton
   - Factory

4. **Параллельные:**
   - Producer-Consumer
