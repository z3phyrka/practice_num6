Observer Pattern

Подпишитесь на уведомления о изменениях в профиле.

Observers: EmailNotifier, SMSNotifier, PushNotifier
Strategy Pattern

Выберите стратегию для экспорта данных профиля.

Стратегии: JSONExportStrategy, XMLExportStrategy, CSVExportStrategy, PDFExportStrategy
Adapter Pattern

Интеграция с социальными сетями через адаптеры.

Адаптеры: FacebookAdapter, TwitterAdapter, GoogleAdapter, GitHubAdapter
Factory Pattern

Создание различных типов пользователей через фабрику.

Фабрики: CustomerFactory, AdminFactory, ModeratorFactory, VendorFactory
